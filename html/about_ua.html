    MatViz3D — це інструмент для моделювання та візуалізації мікроструктур матеріалів з використанням клітинних автоматів. Проєкт був розроблений кафедрою математичного моделювання та інтелектуальних обчислень НТУ "ХПІ" і доступний на GitHub.

    Основні можливості

    1. Візуалізація мікроструктур у форматі 3D для детального аналізу.
    2. Можливість збільшення та повороту куба матеріалу для огляду.
    3. Збереження зображень у різних форматах для документації.
    4. Перегляд гістограми розподілу об'єму зерен.
    5. Анімації генерації мікроструктур.
    6. Вибір алгоритмів для аналізу мікроструктури матеріалів.

    Цільова аудиторія

    1. Матеріалознавці та інженери — дослідники мікроструктур матеріалів для покращення їх властивостей.
    2. Академіки — професори, аспіранти та науковці технічних університетів.
    3. Механіки — професіонали, що аналізують поведінку матеріалів під навантаженням.
    4. Спеціалісти з комп'ютерного моделювання — для 3D-візуалізації та аналізу складних структур.

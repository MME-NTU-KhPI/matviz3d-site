    Спеціальний додаток MatViz3D був розроблений для комп'ютерного моделювання процесу кристалізації та генерації мікроструктур. Основні переваги додатку: тривимірна генерація мікроструктур, що дозволяє дослідникам отримувати детальні та реалістичні зображення структури матеріалу, вибір сусідства клітин для різноманітності генерації та можливість спостереження за поетапним процесом кристалізації. Загалом, MatViz3D є потужним інструментом для моделювання, аналізу та візуалізації мікроструктур матеріалів, використовуючи різноманітні підходи та техніки.

    Інструкції з використання програми

    1. Запустіть додаток
    2. Введіть розмір куба та кількість початкових точок
    3. Виберіть алгоритм для аналізу мікроструктури матеріалу та натисніть кнопку "START"
    4. Ознайомтеся з тривимірною візуалізацією мікроструктури. Збільшуйте та обертайте для детального перегляду
    5. Зберігайте зображення для документації
